1. Data Collection

Green Race collects and stores the following personal data when you save a game:

  • Username: The name you choose to identify yourself in the game
  • Game statistics: Scores, moves, and gameplay data
  • Session data: Temporary data stored in your browser (localStorage)

2. Purpose of Data Processing

Your data is used exclusively for:

  • Displaying game history and statistics
  • Reconnecting you to ongoing games
  • Improving gameplay experience

3. Data Storage

Game data is stored locally on the server running Green Race. No data is shared with third parties.

4. Your Rights (GDPR)

In accordance with the General Data Protection Regulation (GDPR), you have the right to:

  • Access: Request a copy of your stored data
  • Rectification: Request correction of inaccurate data
  • Erasure: Request deletion of your data
  • Portability: Receive your data in a structured format

5. Data Retention

Saved game data is retained indefinitely unless you request deletion. Session data expires when you close your browser.

6. Contact

For any questions or requests regarding your personal data, please contact the administrator of this Green Race instance.

Note: By playing Green Race and saving games, you consent to this data processing.
